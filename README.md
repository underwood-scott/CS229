# CS229 Final Project

To run models:
- Run `get_copernicus_data.py` to download gridded data from Copernicus
- Run `read_data.py` to join the downloaded gridded data to the wildfire database 
and save as csv
- Use `*.ipynb` files to run models using saved dataset
